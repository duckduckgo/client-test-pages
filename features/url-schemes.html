URL Schemes

  • ddgQuickLink://example.com - launches DuckDuckGo app and opens the given URL
  • zoomus://open/1231231231 - should launch the zoom app, but is not a valid zoom link
  • custom://scheme/with.path - a custom URL scheme that probably isn't supported
  • intent://duckduck.com - launches DuckDuckGo Android app or opens the PlayStore
